# Face-Object-Gaze-Proctoring-Model

An advanced AI-powered proctoring system using TensorFlow.js and React.

## Features
- **Identity Verification**: Real-time face matching against a baseline.
- **Object Detection**: Detects unauthorized objects like phones and books.
- **Face Counting**: Alerts on multiple faces or no face in frame.
- **Gaze Tracking**: Monitors attention using facial landmarks.

## Setup
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev`.

## Models Used
- MobileNetV2 (Face Recognition)
- COCO-SSD (Object Detection)
- MediaPipe Face Mesh (Gaze/Lip Tracking)
- MediaPipe Face Detection (Face Counting)
